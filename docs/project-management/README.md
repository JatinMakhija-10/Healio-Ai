# Healio.AI Project Management Setup

Use these files to create a simple command center for the Healio launch.

## Recommended Tools

1. Notion: company command center for marketing, doctor onboarding, launch planning, and daily updates.
2. GitHub Projects: engineering tasks, bugs, pull requests, and release tracking.
3. WhatsApp or Slack: quick team communication and blockers.

## Import Order For Notion

1. Create a Notion page named `Healio Command Center`.
2. Import `marketing_launch_board.csv` as a database.
3. Import `tech_product_board.csv` as a database.
4. Import `doctor_onboarding_crm.csv` as a database.
5. Import `launch_calendar.csv` as a calendar/database.
6. Import `daily_updates.csv` as a database.
7. Copy `weekly_review_template.md` into a Notion page.

## Status System

Use the same statuses everywhere:

- Not Started
- In Progress
- Blocked
- Under Review
- Done

## Priority System

- P0: launch blocker
- P1: important for launch
- P2: should do soon
- P3: nice to have

## Daily Rule

Every team member should update their tasks before the end of the day.

Daily update format:

- Done today
- Blocked by
- Tomorrow
- Help needed

## Weekly Review Rule

Every week, review:

- What shipped
- What is delayed
- Doctor onboarding numbers
- Marketing reach
- Technical blockers
- Launch risks
- Next week priorities
